Our Services

Flu Vaccination Provider Locations

Display the location of flu vaccination providers, across the country. Users will also be able to search by location or zip to find the nearest service providers. Google Maps API will be used to enable this functionality.

Demographics for Flu Vaccination Coverage

Display a bar graph, for flu vaccination rates, for various age groups and races/ethnicities. Users can toggle whether this report is run on the state or national levels, as well as the reporting year. This page may also display a heat map, for the US, to track vaccination rates by month.

Flu Season Vaccination Doses Distributed

Display multiple line graphs, each representing a different year's weekly cumulative vaccine doses distributed, during flu season, throughout the US.

Flu Hospitalization Rates

Will display flu hospitalization rates, per 100,000, for each week of the year. There will be filters for the following options: Allows users to choose to display weekly or cumulative rates. The year the report is being run for. Allows users to choose how the results will be grouped by; age, race/ethnicity, or not grouped.
